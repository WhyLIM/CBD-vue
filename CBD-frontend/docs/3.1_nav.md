# Navigation Overview

## Platform Layout

The CBD3 website is organized into several main sections accessible through the main navigation menu:

### Main Navigation

- **Home**: Platform overview and quick access to key features
- **Biomarkers**: Information about biomarkers
- **Advanced Search**: Search function with additional conditions
- **Explore**: Data explore and analysis
- **Download**: Data download page

### Footer Links

- **Quick Links**: The work unit to which the CBD3 team members belong
- **Help Information**: Help Center Links
- **Technical Support**: Main technical support
