# 浏览与分析

## 生物标志物浏览器

生物标志物页面提供对我们整理的结直肠癌生物标志物数据库的结构化访问。

## 网络探索器

蛋白质相互作用网络

- **网络浏览器**：PPI网络的交互式探索
- **通路整合**：KEGG和Reactome通路
- **置信度过滤**：调整相互作用置信度阈值

## 批量到单细胞分辨率探索器（B2SCR）

### TCGA临床数据

- **患者队列**：COAD和READ数据集
- **临床参数**：人口统计学、分期、治疗
- **分子数据**：表达、突变、甲基化
- **生存数据**：多个生存终点

### GEO单细胞数据

- **可用数据集**：整理的scRNA-seq研究
- **样本信息**：组织来源、处理方法
- **细胞类型注释**：自动和手动标记
- **质量指标**：测序深度、基因覆盖度

## 交互式可视化

### 网络图

- **交互式探索**：缩放、平移和选择网络元素
- **布局选项**：力导向、圆形、层次化
- **节点属性**：大小、颜色和形状编码
- **通路高亮**：可视化生物通路

### UMAP图

- **细胞群体可视化**：细胞类型的二维表示
- **基因表达叠加**：突出特定基因表达
- **聚类识别**：自动和手动聚类
- **细胞类型注释**：交互式标记工具

## 数据导出选项

### 导出格式

- **CSV/Excel**：用于电子表格分析
- **PNG/SVG**：用于出版就绪的图形
