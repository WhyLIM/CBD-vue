# 数据访问

CBD3提供多种方式来访问和下载我们全面的结直肠癌生物标志物数据。

## 访问方式

### Web界面

- **交互式浏览**：用户友好的数据探索Web界面
- **高级搜索**：具有多重过滤器的强大搜索功能
- **可视化工具**：用于数据分析的交互式图表和图形
- **导出选项**：以各种格式下载结果

### API访问

- **RESTful API**：对所有CBD3数据的程序化访问
- [ ] API文档正在建设中

## 访问级别

- **无需注册**：基本浏览和搜索

- **完全访问**：目前无限下载和API访问

## 数据使用协议

- **研究目的**：学术和非营利研究
- **教育用途**：教学和学习活动
- **个人分析**：个人研究和学习
- **开源项目**：在开源工具中集成数据
