# 介绍与概述

## 介绍

欢迎使用 CBD3（结直肠癌生物标志物数据库 v3），这是一个通过生物标志物、临床数据和单细胞RNA测序数据的综合分析来加速结直肠癌研究的综合平台。

我们的平台为研究人员提供强大的工具来：

- 包含从精选医学文献中整理并手动验证的结直肠癌生物标志物
- 可视化和探索单细胞RNA测序数据
- 研究蛋白质-蛋白质相互作用网络
- 访问整理的临床试验数据和患者结局
- 跨多种数据类型执行高级搜索

## 快速开始

1. **探索生物标志物**：浏览我们全面的结直肠癌生物标志物集合
2. **分析数据**：使用我们的交互式工具探索蛋白质-蛋白质相互作用网络和单细胞图谱
3. **可视化结果**：创建研究发现的定制可视化
4. **导出数据**：以各种格式下载结果进行进一步分析

## 数据来源

我们的数据库整合了来自多个权威来源的数据：

- **PubMed文献**：来自科学文献的整理生物标志物数据
- **TCGA**：来自癌症基因组图谱的临床验证数据
- **GEO**：来自基因表达综合数据库的单细胞RNA测序数据
- **STRING**：来自STRING数据库的蛋白质-蛋白质相互作用网络

## 技术栈

- **前端**：Vue 3, Pinia, Element Plus
- **后端**：Express.js, MySQL
- **可视化**：Echarts, Cytoscape.js
- **文档**：VitePress
