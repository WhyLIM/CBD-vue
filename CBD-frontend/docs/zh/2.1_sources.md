# 数据来源

CBD3整合了来自多个权威来源的数据，为结直肠癌生物标志物研究提供全面的资源。

## 主要数据来源

### 1. PubMed文献数据

我们的生物标志物数据库主要通过PubMed文献构建：

- **自动收集**：系统性地挖掘与结直肠癌生物标志物相关的PubMed摘要和全文文章
- **手动验证**：专家审查提取的生物标志物以确保临床相关性和准确性
- **定期更新**：年度更新纳入最新发表的研究（实际上，我们正在尝试自动化该流程以提高更新频率）

**数据包含**：

| 字段名称     | 详细描述                                          |
| ------------ | ------------------------------------------------- |
| ID           | 每条记录的唯一标识符                              |
| Title        | 研究文章的标题                                    |
| Authors      | 研究的所有作者                                    |
| First_Author | 负责该工作的主要作者                              |
| Year         | 研究的发表年份                                    |
| Abstract     | 研究目标、方法、结果和结论的摘要                  |
| Keywords     | 用于索引和搜索文章的术语                          |
| PMID         | PubMed数据库中的唯一标识符                        |
| DOI          | 文章的持久数字标识符                              |
| PMC          | PubMed Central中免费全文文章的标识符              |
| Journal      | 发表的科学期刊名称                                |
| Biomarker    | 用作生物状态或条件指标的生物分子                  |
| Type         | 生物标志物的分类（如蛋白质、基因、代谢物）        |
| Category     | 一般分类组                                        |
| Symbol       | 官方基因符号（如TP53）                            |
| Description  | 生物标志物功能和特征的详细说明                    |
| Region       | 研究进行的地理位置                                |
| Race         | 研究参与者的种族或种族背景                        |
| Number       | 研究中参与者或样本的总数                          |
| Male         | 男性参与者数量                                    |
| Female       | 女性参与者数量                                    |
| Gender_M/F   | 男性与女性参与者的比率或分布                      |
| Age_Mean     | 研究参与者的平均年龄                              |
| Age          | 参与者的年龄范围或分布                            |
| Location     | 肿瘤或疾病的解剖位置                              |
| Stage        | 疾病的临床分期（如癌症分期）                      |
| Source       | 使用的生物样本类型（如组织、血液、血清）          |
| Experiment   | 用于生物标志物检测的实验方法（如IHC、PCR、ELISA） |
| Statistics   | 用于数据分析的统计方法                            |
| Application  | 生物标志物的临床或研究应用                        |
| Conclusion   | 研究的主要发现和结论                              |
| Clinical_Use | 潜在的临床用途或应用                              |
| Target       | 治疗或干预的分子靶点                              |
| Drugs        | 与生物标志物相关的药物或治疗剂                    |

### 2. TCGA临床验证数据

临床验证数据来源于癌症基因组图谱（TCGA）：

- **患者队列**：COAD（结肠腺癌）和READ（直肠腺癌）数据集
- **临床参数**：患者人口统计学、肿瘤分期、治疗结果
- **生存数据**：总生存期、无病生存期、无进展生存期

### 3. 单细胞图谱

单细胞转录组数据来源于基因表达综合数据库（GEO）。

### 4. 蛋白质相互作用组

蛋白质-蛋白质相互作用数据整合自STRING数据库。

### 5. 通路数据库

- **COMPARTMENTS**：亚细胞定位数据库
- **TISSUES**：组织表达数据库
- **DISEASES**：疾病关联数据库
- **Process**：基因本体生物过程
- **Component**：基因本体细胞组分
- **Function**：基因本体分子功能
- **Keyword**：UniProt关键词
- **KEGG**：京都基因与基因组百科全书
- **SMART**：蛋白质域注释数据库
- **InterPro**：蛋白质家族分类数据库
- **Pfam**：蛋白质家族数据库
- **PMID**：PubMed文献数据库
- **RCTM**：Reactome通路数据库
- **WikiPathways**：社区整理的通路数据库
- **HPO**：人类表型本体
- **NetworkNeighborAL**：STRING功能蛋白质关联网络

### 6. 药物数据

药物数据和靶点识别是从DrugBank提供的XML文件解析的。

## 数据许可

CBD3中使用的所有数据来源都符合以下要求：

- **开放获取**：优先选择开放获取的数据来源
- **研究用途**：许可用于研究目的的数据
- **归属**：正确归属原始数据提供者
- **合规**：遵守数据使用条款和条件
