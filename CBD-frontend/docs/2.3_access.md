# Data Access

CBD3 provides multiple ways to access and download our comprehensive colorectal cancer biomarker data.

## Access Methods

### Web Interface

- **Interactive Browsing**: User-friendly web interface for data exploration
- **Advanced Search**: Powerful search functionality with multiple filters
- **Visualization Tools**: Interactive charts and graphs for data analysis
- **Export Options**: Download results in various formats

### API Access

- **RESTful API**: Programmatic access to all CBD3 data
- [ ] API documentation is under construction

## Access Levels

- **No Registration Required**: Basic browsing and searching

- **Full Access**: Currently unlimited downloads and API access

## Data Use Agreement

- **Research Purposes**: Academic and non-profit research
- **Educational Use**: Teaching and learning activities
- **Personal Analysis**: Individual research and study
- **Open Source Projects**: Data integration in open source tools
