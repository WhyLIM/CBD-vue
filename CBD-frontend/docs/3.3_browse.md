# Browse & Analysis

## Biomarker Browser

The Biomarker Page provides structured access to our curated colorectal cancer biomarker database.

## Network Explorer

Protein Interaction Networks

- **Network Browser**: Interactive exploration of PPI networks
- **Pathway Integration**: KEGG and Reactome pathways
- **Confidence Filtering**: Adjust interaction confidence thresholds

## Bulk to Single Cell Resolution Explorer (B2SCR)

### TCGA Clinical Data

- **Patient Cohorts**: COAD and READ datasets
- **Clinical Parameters**: Demographics, staging, treatment
- **Molecular Data**: Expression, mutation, methylation
- **Survival Data**: Multiple survival endpoints

### GEO Single-Cell Data

- **Available Datasets**: Curated scRNA-seq studies
- **Sample Information**: Tissue source, processing method
- **Cell Type Annotations**: Automated and manual labeling
- **Quality Metrics**: Sequencing depth, gene coverage

## Interactive Visualizations

### Network Graphs

- **Interactive Exploration**: Zoom, pan, and select network elements
- **Layout Options**: Force-directed, circular, hierarchical
- **Node Attributes**: Size, color, and shape encoding
- **Pathway Highlighting**: Visualize biological pathways

### UMAP Plots

- **Cell Population Visualization**: 2D representation of cell types
- **Gene Expression Overlay**: Highlight specific gene expression
- **Cluster Identification**: Automated and manual clustering
- **Cell Type Annotation**: Interactive labeling tools

## Data Export Options

### Export Formats

- **CSV/Excel**: For spreadsheet analysis
- **PNG/SVG**: For publication-ready figures
